# UNIQLO Demo (Split Pages)

## Chạy nhanh
- Mở `index.html` (nếu dùng VSCode Live Server thì càng tốt).

## Trang
- `index.html` : Trang chủ
- `about.html` : Giới thiệu
- `products.html` : Sản phẩm (dropdown Nam/Nữ + bộ lọc)
- `promo.html` : Khuyến mãi
- `contact.html` : Liên hệ
- `product-detail.html?id=1` : Chi tiết sản phẩm
- `checkout.html` : Thanh toán

## Lưu ý
- Giỏ hàng được lưu bằng `localStorage`, nên đổi trang vẫn còn.
- Ảnh vẫn dùng đường dẫn cũ `assets/image/...` (bạn copy nguyên thư mục assets của bạn vào đây).
